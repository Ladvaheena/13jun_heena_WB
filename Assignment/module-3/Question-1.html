Qu:-1 What are the new tags added in HTML5?

Answer:
↦ There are to types of tags in HTML-5.
  1. Structural tag or Sementic tag
  2. Content tags

1.Sementic tags

↦ Sementic element also known as "structural element".it creates its own structure.
↦ Semantic elements means 'elements with a meaning,which specify its meanings towards browers & developers.
↦ List of such type of elements given below;
  1. header tag: specifies header of the webpage or document.
  2. nav tag: specifies navigation bar or link of the webpage or document.
  3. section tag
  4. article tag
  5. Summary tag
  6. footer tag: specifies footer part of the webpage or document.

2.Content tags

↦ Content tags used in content part of the html file.
↦ Some of this,list given below;
  1. Date: apply date,it is input type of in form .
  2. Color:apply color
  3. Video: Embed video in HTML file.
  4. Audio: Embed audio in HTML file.
  5. Del: used to show text deteled.
  6. Progess: used to show process line.
  7. Poster:used to apply image in embed video,which is show before start the video.
  8. sub: used to write text in subscript
  9. sup: used to write text in superscript.
  10. pre:used to show same output of content in website as well as in code.
