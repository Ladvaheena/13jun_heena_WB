Qu:-18 What is SVG?

Answer :
↣ SVG- Scalable vector graphics
↣ SVG is used for create graphics for web page &web sites.
↣ SVG gives best output as compare to canvas graphic designer.
↣ SVG creates graphics without use of JAVA Script.
↣ SVG is a vector based image formate to represents two dimensional graphics.
↣ With help of the SVG we can create lines,shapes or curves.
