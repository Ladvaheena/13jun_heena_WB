Qu:-13 What is the use of an iframe tag? With Example.

Answer:
↣ The <iframe> tag in HTML is used to embed another HTML document or web page with in the current document.
↣ The "src"attribute in iframe tag used to give path of other HTML document or web page.
↣ We can set width,height and border of the document.
↣ This tag commonly used for to give advertisement,embedded vedio,image,web analytics etc.. to web page.
↣ Example of it given below..

<iframe src="file name" width="" height="" name="" > </iframe>
