Qu:-16 How are active links different from normal links?

Answer :
Normal link:
Normal link is just line of code that contains a pointer to another resource.
Normal link are simple type of URL (hyperlink),which can be apply to text or image.
In html,tag anchor(a) used to create text or image hyperlink.
Active link:
Active link is that line of code in action,opening in another resource.
A link become active, when user click on the link.
Active link shows that the browser is in process to load a new resource.
Active link might run when the user pressrs Enter in a specific field or clicks a button on the form.
