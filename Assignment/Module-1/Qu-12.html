Qu:-12 How to create a Hyperlink in HTML? With Example.

Answer:
↣ To create a hyperlink, Anchor element (<a>) with "href" attribute can be used.
↣ With the use of this element ,we can apply URL link to image or text.
↣ Simple formate of this element given below,

<a href="URL link" > text/image </a>
