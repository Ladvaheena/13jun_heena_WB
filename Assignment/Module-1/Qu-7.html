Qu:-7 What is the difference between the ‘id’ attribute and the ‘class’ attribute of HTML elements? With Example.

Answer :
↣ The basic Difference between the 'id' attribute and the 'class' attribute that "id attribute" can apply to only one element in one web page,  where as the "class attribute" can apply to more then one element in a single web page.
↣ In simple words,id attribute must be unique ,it can't has same value for element in a single web page.
↣ Where as, class attribute should be same value for difffrent element and diffrent name value for single element.
↣ Example of this element given below..

<h2 class="redbg black"> Use of class attribute </h2>
↣ in above examples,"redbg" and "black" is the name of the class,& "class" is a attribute.
↣ Here "redbg" and "black" both class value used for same element h2.

<h2 id="HTML File"> Use of id attribute </h2>
↣ in above examples,"HTML File"is the name of the id,& "id" is a attribute.
