Qu:-15 How to insert a picture into a background image of a web page? With Example.

Answer :
↣ Simplely we insert a picture inta a background image with use of background image property in internal CSS.
↣ We can pass image URl in 'div' attribute.
↣ We can set image height & size.
↣ Simple formate to apply background image is given below..
<head>
<style>
div{background-image:url( pass path of the image);
background-size: set size;
height:set height;}
</style>
</head>
