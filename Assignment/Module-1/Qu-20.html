Qu:-20 What are logical and physical tags in HTML?

Answer :
Logical Tag :
Logical tags are used tell the brower what kind of text is written inside the tags.
Logical tags provide any information about the text inside the tags.
Logical text are used in HTML to display th text according to the logical styles.
Ex:<strong> ↣ This tag used to tell browser that this text is important.
Physical Tag:
Physical tags are used to provide the visual appearence to the text.
Physical tags do not provide any information about the text.
Physical Text Styles indicate the specific type of appearance for a section e.g., bold, italics, etc.
Ex:<b> ↣ This tag used for bold the text.
