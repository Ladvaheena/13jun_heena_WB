Qu:-6 What is the ‘class’ attribute in HTML? With Example.

Answer :
↣ The class attribute specifies one or more class names for an element.
↣ The class attribute can be used for any HTML element.
↣ This attribute is case sensitive.
↣ We can use same class name with different elememt.
↣ We can apply different class name to one element by seperating them with space.
↣ Example of this element given below..

<h2 class="redbg"> Use of class attribute </h2>
↣ in above examples,"redbg"is the name of the class,& "class" is a attribute.
