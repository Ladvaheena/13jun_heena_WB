Qu:5 What is the difference between Bootstrap 4 and Bootstrap 5

Answer:

Bootstrep-4 Bootstrep-5
In Bootstrep-4,it has jQuary & all related plugins. In Bootstrep-5,JQuary is removed & switch to javascript.
It has limited colors. Extra color will be added.
It supports IE 10 & 11 both internet explorer. It does not support IE 10 & 11 both internet explorer.
In this version,user can't modify the utilities. In this version, user can modify the utilities & also create own utilities.
In this,user can use gutter property with fontsize in px. In this,user can use gutter property with fontsize in rem.
It does not have own SVG icons. It has own SVG icons.
This version supports jumbotron. This version does not support jumbotron.
In navbar,user will get white dropdown as default In navbar,user will get black dropdown as default
Bootstrap 4 uses Jekyll software. Bootstrap 5 uses Hugo software as it is fast static site generator.
