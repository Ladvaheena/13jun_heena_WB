Qu:-3 What are the default Bootstrap text settings?

Answer:

->In bootstrep-5 version,the default font-size is 16px and line-height is 1.5
->In bootstrep-4 version,the default font-size is 16px and line-height is 1.5 & font-family is "Helvetica Neue, Helvetica, Arial, sans-serif".
->In bootstrep typography,various text-formating class are available,which is listed below.
  1. h1toh6 class : This is use of h3 class
  2. display1 to display6 class: This is use of display 6 class
  3. mark class: this use of mark class
