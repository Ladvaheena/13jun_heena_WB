Qu:8 In Bootstrap 4, what is flexbox?

Answer:

-> In bootstrep-4 flex box is one type of layout modual makes easy to create flexible reponsive layouts without use of float property.
-> It gives same worksas well as display-flex property of CSS works.
->Use of flexbox,d-flexclass apply to the elemnt. & flexbox is worked.
->Here, list of some property of flexbox.
  • .flex-row : set flex item in horizontal.
  • .flex-row-reverse : set flex item in horizontal align but opposite side.
  • .flex-column : set flex item in column.
  • .flex-column-reverse : set flex item in column align but oppsite side.
  • .justify-content : set flex item alignment in horizontal.
  • .align-item : set flex item alignment in vertical.
  • .align-self : set individually flex item alignment in vertical.
