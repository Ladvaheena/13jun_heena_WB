Qu:10 What is a bootstrap card and how would you create one?

Answer:

->Bootstrep card is one type card,whose simple layout is provided by bootstrep.
->Main "card -class"is apply to parent element.
->"card-img"class for image in card.
->"card-body"class for card body.
->Inside the card body,"card-title" class for card title."card-text" class for card-text & "card-footer" class for adding foorter in card.
->User can also add links,list groups & button as per requriment.
