Qu:- 4 Name a few CSS style components.

Answer:
-> CSS style include two Components,
1.Property
2.Value
->Above,both components parts of the CSS Syntax.
->With use of this two components, we can apply style to the webpages.
->For Example:if we want to apply color of this div tag,for this color propertyis used and then assign the value to the property. 
        <style>
             div{
                color: blue
             }
        </style>
        -> here color is property name & blue is value.
