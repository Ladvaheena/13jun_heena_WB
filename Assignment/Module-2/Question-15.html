Qu:-15 What are the external style sheets?

Answer:

An External Style Sheet

↦ An external style sheet is a separate CSS file with .css extension , which have various style rule for HTML element.
↦ This style sheet one of the three ways to integrate css in web pages.
↦ External css apply with use of href & rel attribute in link tag in head tag.
↦ In href attribute, pass the path of css file.
↦ In rel attribute, apply "stylesheet" value.
↦ Benefits of this type,we can apply style to web page through one file.
↦ Syntax: 

            <head >
            <link rel="stylesheet" herf="path of externalstylesheet">
            </head >
