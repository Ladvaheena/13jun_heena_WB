Qu-5:- What do you understand by CSS opacity?

Answer:
↦ CSS Opacity property used to set opacity or transparency of whole element including background-color or text.
↦ If we want to apply particular only one element,we can also set that,just apply to the particular content tag.
↦ It is apply by Fliter property and set the value.
↦ The value can be set either in number or percent.
↦ In number value set between 0-1; 0 number shows fully transparent and 1 number shows original element.
↦ In percent value set between 0% to 100%; 0% number shows fully transparent and 100% shows original element.
