Qu:-19 What is the rule set?

Answer:

Rule Set

↦ A rule set is aon type of statement ,which tells the browers how to show the HTML element in the webpages or web site
↦ Rule set in CSS have two parts ,
  1. Selectors-------- HTML element,on which user want to apply CSS; for example -- h3 tag, div tag, class & id name etc..
  2. Declaration Box------- Various Css style; for example -- color:red , text-align:center,display: inline etc..
↦ Declaration box start with left curly bracket({) and end with the same right curly bracket(}). in b/w two curly brackets, there are list of various CSS style declarations.
↦ Various style declarations b/w two curly brackets separated by "semicolon(;)" symbol.
↦ Declaration has Css property name,followed by colon(:) symbol & then apply property value.
↦ In simple way, CSS Rule Set present the Syntax of Css.
↦ Syntax: 
    Selector/element name{
        property-name1:property-value1;
        property-name2:property-value2;
    }
