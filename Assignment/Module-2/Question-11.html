Qu:-11 How to center block elements using CSS1?

Answer:
↦ Block level elements are those, which takes full width of the screen and takes height as per the content.
↦ There are the various block level element, like h1 to h6 tags , p-tag, div tag, ul &li tag etc..
↦ To center the block level elements use "text-align" property and give "center" value.
