Qu:-6 How can the background color of an element be changed?

Answer:
↦ For the change of background-color of an element background-color CSS property is used.
↦ We can apply this property by inline,internal or external style sheet.
Syntax of background-color:
element {background-color:colorname;}------- for internal or external style sheet.
<element style="background-color:colorname" >-------- for inline style sheet.
