Qu:-14 What is embedded style sheets?

Answer:

↦ Embedded style sheets are one type of external style sheet, when its attach in head tag in link tag, then it's called embedded style sheet.
↦ Syntax : 

        <head >
        <link rel="stylesheet" herf="path of externalstylesheet">
        </head >

↦ With use of the Embedded style sheet,we can apply css to whole webpages by just attaching one file in head tag.
