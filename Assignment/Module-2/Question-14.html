Qu:-14 What is embedded style sheets?

Answer:

↦ Embedded style sheets are internal style sheet,which is apply in head tag of HTML file.
↦ Generally, css style apply to particular tag with use of various selector;like..Class, id, element etc..
↦ Syntax : 

        <head >
            <style> 
                 Write CSS style in curly bracket{}
            </style>    
        </head >

↦ With use of the Embedded style sheet,we can apply css into HTMLfile in head tag.
