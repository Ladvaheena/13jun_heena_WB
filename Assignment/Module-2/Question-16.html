What are the advantages and disadvantages of using external style sheets?

Answer:
External Style Sheet
↦ An external style sheet is a separate CSS file with .css extension , which have various style rule for HTML element.
↦ Syntax: 
                   <head >
                   <link rel="stylesheet" herf="path of externalstylesheet">
                   </head >
Advantages of external style sheet
↦ With Use of this type of style sheet,we can apply style to all web page through one file.
↦ Morever,We can use same Css style file for diffrent HTML file,that means an external style sheet can be reused across multiple pages and websites.
↦ Therefore, only changes in one css file, shows changes in all linked file.
↦ The Reuseble ability makes it time saving & provides easy maintainance to developer.
Disvantages of external style sheet
↦ With use use of this CSS style,users should maintain the both file, 1.html file & 2. CSS File. These may be create some confusion & extra work.
↦ User's web pages don't show/work properly,until the External Style Sheet is loaded.
↦ If users apply same external style sheet to multiple HTML file,it takes more time for download user's website.
