Qu:-13 What are the ways to integrate CSS as a web page?

Answer:

↦ CSS stands for the Cascading Style Sheet.

↦ CSS is used to apply style in webpages or website,which is making beautiful & attractive towards client & users.
↦ CSS is Markup Lauguage.
↦ There are three ways to integrate CSS in webpages,which is listed below..

  1. Inline CSS
  2. Internal CSS
  3. External CSS

1.Inline Css
↦ Inline css apply in particular tag/element in style attribute.
↦ It gives output only those tag in which inline css style applied.
↦ It has more importance than other css stylesheet.

2.Internal Css
↦ Internal css apply by style tag in head tag.
↦ Generally, css style apply to particular tag with use of various selector;like..Class, id, element etc..
↦ Css style rule apply in Curved brekets.

3. External Css
↦ External css apply with use of href & rel attribute in link tag in head tag.
↦ In href attribute, pass the path of css file.
↦ In rel attribute, apply "stylesheet" value.
↦ Benefits of this type,we can apply style to web page through one file.
