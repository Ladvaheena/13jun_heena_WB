Qu:-8 What is the use of the background-position property?

Answer:
↦ The use of background-position property, We can set starting position of the background-image.
↦ By default,background-position is top-left.
↦ We can apply background-position by these ways,
By apply left,right,top or bottom as a value in background-position property.
CSS Syntax: {background-position:top-left}
