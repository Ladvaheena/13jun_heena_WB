Qu:-1 What are the benefits of using CSS?

Answer:

CSS stands for Cascading Style Sheet

↦ Main work(use) of the CSS is apply various style to the text,image or all content part of the webpages or websites.
↦ By using CSS style,we can make webpages or websites more effective & attractive.
↦ Some benefits using CSS is listed Below..
Benefits of using CSS
  • Main benefit of using CSS is that,it makes webpages's content more stylish & effective.
  • It is time saving property.
  • With the use of external style sheet,we can apply css to all element of web page.
  • We can use same external style sheet for different HTML files.
  • CSS style provides easy formatting changes by using various style components.
