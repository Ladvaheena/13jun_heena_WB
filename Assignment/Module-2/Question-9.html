Which property controls the image scroll in the background?

Answer:
↦ For controls the image scroll in the background, the background-attachement property used.

◉ Background-attachement property

↦ Background-attachement property has two value.
  • Scroll value---- It's set by default value, background scroll with the page.
  • Fixed value---- background become fixed,not scroll with the page.
