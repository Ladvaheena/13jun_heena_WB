Qu:-7 How can image repetition of the backup be controlled?

Answer:
↦ With use of background-repeat property, We can control the background-image repetition.
↦ Syntax of background-repeat property given below...
element{background-repeat:no-repeat}
↦ This property has diffrent value,which is listed below..
  1. background-repeat: repeat -- By default set this value, image show repetation.
  2. background-repeat:no-repeat-- Image does not show repetation
  3. background-repeat:repeat-x-- Image show repetation on top side.
  4. background-repeat:repeat-y-- Image show repetation on left side.
