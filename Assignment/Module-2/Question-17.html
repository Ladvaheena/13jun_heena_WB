Qu:-17 What is the meaning of the CSS selector?

Answer:
↦ The CSS Selector selects the HTML elements ,on which we want to apply Various css style .
↦ There are various types of Selectors.
  1. Simple Selector
  2. Combinator Selector
  3. Pseudo-class Selector
  4. Pseudo-element Selector
Simple Selector
↦ In this type of selector,select elements by it's name,id & class.
↦ Element selector:Element selector used to apply style in particular element.Element selector apply with name of element in style tag on head tag.
↦ Class selector: Class selector used to apply style in particular class name.Class selector apply with fullstop mark(.) & then written class name.
↦ Id selector: Id selector used to apply style in particular id name.Id name must be unique.Id selector apply with hash symbol(#)& then written id name.
Combinator Selector
↦ In this type of selector, Select element based on their relationship between them.
↦ Group selection: Group selection used to apply css style to particular group of the element.For apply group selection, write element in style tag by separated them with coma (,) symbol.
↦ Child Selector: Child Selector used to apply to css internal(child) element of main(Parent) element.This selector apply by write parent element ,then graterthan (>) symbol & child element.
↦ General selector:General selector used to select the group of element that share the common parent element.This selector apply by element-1 tlide (~) symbol & element-2
↦ Adjacent Selector:The adjacent selector is used to select an element that is directly after another specific elements. Adjacent meaning Immediately following.Sibling elements must have the same parent element.This selector apply by element-1 plus(+)symbol & element-2.
Pseudo-class selector
↦ pseudoclass selector apply with colon (:) symbol.
↦ hover selector:hover selector is used to apply(change) style on various element,when mouse over them.hover selector not supported for link.We can change cursor style with the use of cursor property.
↦ Focus Selector:It is used to apply (change) style on element,when click on them.It is apply by colon(:) symbol with write focus.
↦ Link selector:It is used to apply (change) style on hyperlink.We can change style diffrent css to various three types of links:1. Normal link   2.Active link   3.Visited link
Pseudo-element Selector
↦ Pseudoelement selector apply with Double colon (::) symbol.
↦ After - Before Selector:After & before Selector used to apply css after& before of the content. In both selector, Content Property must be appiled.Without content property it's doesn't work. In content Property,value must be apply in single coma ('').
↦ Firstline/letter Selector:Firstline/letter selector used to apply css in firstline/letter of the element content. It is apply by this way: (element::firstline/letter{css style}) in external or internal style sheet.
↦ Placeholder selector:Placeholder is one type of attribute used in creation of the login form.Placeholder selector used to apply css to placeholder attribute.It is apply by this way:(::placeholder{css style})in external or internal style sheet.
