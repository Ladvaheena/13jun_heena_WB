Qu:-7 What is === operator?

Answer:
-> === operator is one type of comparison operator which is called "strict operator"
-> === operator used to compare equal value of operand & return boolen value means true or false.
-> This === operator return true, if value & type of operand are equal.
-> Example of given below, 
        <script>
        let a = '5',b = 5;
        console.log(a===b)
        Here, value are same but type is not equal ,therefore it's return false.
        let c = '3', d ='3';
        console.log(c===d)
        Here, value & type both are same,therefore it's return true.
        </script>
