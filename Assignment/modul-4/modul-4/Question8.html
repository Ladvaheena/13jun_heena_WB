Qu-8 How can the style/class of an element be changed?

Answer:
->In javascript, commonly two methods are used to change style/class of an element.
1.style.property
2.Changing the class itself

1.style.property

-> this property is used to change the style of an element.
->Syntax:document.getElementById("id").style.property = new_style ->Example;
    <script>
    function stylechange(){
        document.getElementbyId("btn").style.color = "red"
    }
    </script>

2.Changing the class itself

-> This method is used to change class of the element.
->Syntax:document.getElementById("id").className = class
->Example;
    <script>
    function stylecolor(){
        let btn = document.getElementById('color')
        btn.className = "black"}
    </script>
