Qu:-14 How can a page be forced to load another page in JavaScript?

Answer:
->In JavaScript, window.location object to force a page to load another page.
->In which,the location object is used to set the URL of a new page.
->there are two methods.
1.window.location.replace() method
2.window.location.assign() method
->above two method gives same output but main difference between them is in window.location.assign()method the priveous web page is stored in brower's history, so user can use that page again.
-> In window.location.replace() method ,the replced web page not save in brower's history. so user can not used previous web page again.

1.window.location.replace() method

->syntax:window.location.replace("new_url");
->Example;
            <script>
            function forceload() {
                window.location.replace("https://www.youtube.com");
             }
            </script>

2.window.location.assign() method

->syntax:window.location.assign("new_url");
->Example;
            <script>
            function force() {
                window.location.assign("https://www.google.com");
             }
            </script>
