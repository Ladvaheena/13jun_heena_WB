What is the function of the delete operator?

Answer:

* Delete operator

-> delete operator used to remove/delete property from an object.
-> On deletation of property,it will return true,if property does not remove/detele it will return false.
-> If we will try to delete undeclared property , it will also return the true value.
->Syntax: delete object.property
->One example of it given here,
    <script>
    let Employee = {
        firstname: 'Pooja',
        lastname: 'Sharma',
      };
      console.log(Employee.firstname);
      return output: "John"
    //now,delete the firstname..
      delete Employee.firstname;
      console.log(Employee.firstname);
      return output: undefined------ because,now firstname property will be deleted.
    </script>
