Qu:-3 What is negative Infinity?

Answer:

*Negative Infinity

->Negative Infinity is "a number lower than any type of number"
-> Negative Infinity is a special numerical value.
-> If We divided any negative value by Zero than,its will gives output as "negative Infinity"
-> -infinity in javascript is a valid number,therefore if we check the type of it.then it will show a number.
*Syntax:Number.NEGATIVE_INFINITY
-> Example of it given below,

            <script>
