How can you convert the string of any base to an integer in JavaScript?

Answer:
->There are two method to convert the string to an integer.
1.parseInt()
2.parseFloat()

1.parseInt()


-> This is used to convert the string to an integer.
-> If the given value is not convertible it gives NaN, other cases it converts string into integer.
-> With use of this ,we not convert any desimal value into a integer.
-> Syntax:parseInt('string value');
-> Example of it given here,
        <script>
        let a = parseInt('56');
        console.log(a);
        //output :- 56
        let b = parseInt('5.6');
        console.log(b);
        // output :- 5 ---- because ('5.6') is desimal number
        let c = parseInt('hello');
        console.log(c);
        //output:- NaN ---- because ('hello') is not convertible number.
        let d = parseInt('56hello');
        console.log(d);
        //output:- 56 ---- because ('hello') is not convertible number or 56 is convertible.
        </script>
-> from above example,this converts value into integer till it's convertible.

1.parseFloat()


-> This is used to convert the string to an integer.
-> If the given value is not convertible it gives NaN, other cases it converts string into integer.
-> With use of this ,we also convert any desimal value into a integer.it is the main difference b/w parseInt() & parseFloat()
-> Syntax:parseFloat('string value');
-> Example of it given here,
        <script>
        let a = parseFloat('56');
        console.log(a);
        //output :- 56
        let b = parseFloat('5.6');
        console.log(b);
        // output :- 5.6
        let c = parseFloat('hello');
        console.log(c);
        //output:- NaN ---- because ('hello') is not convertible number.
        let d = parseFloat('56hello');
        console.log(d);
        //output:- 56 ---- because ('hello') is not convertible number or 56 is convertible.
        </script>
-> from above example,this converts value into integer till it's convertible.
