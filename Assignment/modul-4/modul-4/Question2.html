*isNAN function

-> isNan function, NaN is stands for the 'Not A Number'.
-> According to its name,isNaN() function used to check that the apply value is either NaN (not a number)means alphabet,special character or not.
->The Syntax of the isNaN function is "isNaN(value)".
->The isNaN() gives true output, if the value is NaN,and gives false output, if the value is number.
