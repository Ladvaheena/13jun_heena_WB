Qu:-1 What is JavaScript?

Answer:

-> JavaScript is a scripting lauguage ,which makes webpages or website more interactive with user.
-> Javascript is used to make websites more responsive towards users.(client side).
-> Javascript is different from "JAVA" because, JavaScript is scripting lauguage,where as java is programming lauguage.
-> JavaScript is client side scripting lauguage and it is case sensitive.
-> With use of javascript ,we can make Drop-down menu,various maths operation& validation of form etc...
-> In two ways, we can apply javascript in HTML file.
1.Internal JavaScript:
* Internal javascript pass through < script > tag in head tag or just before complete the body tag.
2.External JavaScript:
* External Javascript is a separate file of javascript code,save it with ".js" extension .
* External Javascript file pass path of the file through < script > tag in head tag or just before complete the body tag.
