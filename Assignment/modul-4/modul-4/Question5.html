Qu:-5 What are undeclared and undefined variables?

Answer:

*Undeclared Variable
->Undeclared variable are those ,that do not exist in the programme & are not declared.
->if we try to read the value of the undeclared variable , then at runtime , error is showed in console.
->Example of it given below,
            <script>
            console.log(myvariable)
            </script>
            At run time show error:-  //ReferenceError:myvariable is not defined.


*Undefined variables
-> Undefined Variable are those,which are declared but not assign any type of value to it .
-> If we try to read the value of this undefined variables ,"undefined" is showed.
->Example of it given below,
            <script>
            let myname;
            console.log(myname)
            </script>
            At run time show :-  // undefined.
