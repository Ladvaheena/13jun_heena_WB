How to Apply CSS Using JQuery, How to Add Class and Remove Class in Jquery , JQuery Animation?

Answer:

1.Apply CSS

-> For Apply CSS Using JQuery,Css() method is used.
-> for Apply jQuary to the web page,the link of the jQuary must be apply in script tag.
example: 
         <script >
            $(document).ready(function () {
                $('pre').css('background', 'skyblue'))}

2.Add class & remove class

-> for add the class .addclass()method is used.
-> for remove the class .removeclass()method is used.
-> if we want to apply add or remove class together toggleclass()methgod is used.
*syntax: 
            <script >
            $(document).ready(function (){
                $("p").addClass("bgcolor");-----for add class
                $("p").removeClass("bgcolor")----for remove class
                $("p").toggleclass("bgcolor")---- for add/remove process together)}
            </script >

3.Animation JQuary

-> for create Animation using JQuary animate() method is used. *syntax: 
            <script >
            $(selector).animate({params},speed,callback);
            <script >
