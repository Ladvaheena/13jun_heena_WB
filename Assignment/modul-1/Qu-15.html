Qu:-15 How to insert a picture into a background image of a web page? With Example.

Answer :
↣ Simplely we insert a picture inta a background image with use of background image property in internal style sheet.
↣ In background-image property,pass the path of background-image in "url"attribute.
↣ We can set image size by background-size property.
↣ We can also apply by inline css in particular element with use of style attribute & use background-image property.
↣ Simple formate to apply background image is given below..
    <head>
    <style>
          elemnt-name{background-image:url( pass path of the image);
              background-size: set size;
          }
    </style>
    </head>
↣ For example; here,background-image appled in body tag by internal CSS.
