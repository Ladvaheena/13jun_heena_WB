Qu:-2 What are tags and attributes in HTML?

Answer:-Tags:The tags are the keyword on a web page that define how your web brower must format and display your web page.
↣There are Two types of tags:  1. Paired Tag  2.Non Paired Tag
paired tag : This Tag must be close.
Example:<title>  </title>
Nonpaired tag:this tag not close.
Example:<br> and <hr>  etc..
 
Attributes: Attributes provied additional information about element.
↣Attributes always used in the start tag,
↣Attributes can be used to change color,size and functionally of html element.
↣Attributes usually come in name/value pair like:name="value".
