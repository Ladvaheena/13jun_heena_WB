Qu:-19 What is difference between HTML and XHTML?

Answer :
↣ HTML: Hypertext Markup Language.
↣ XHTML: Extensible Hypertext Markup Language.

↣ some difference between HTML and XHTML given below..

Diffrence b/w HTML & XHTML
HTML XHTML
HTML tags are not Extensible. XHTML tags are Extensible.
HTML uses a formate that is similar to document formate. XHTML uses markup language.
HtML is not case sensitive. Where as, XHTML only allowed lower case.
HTML can use open tag,like; <br>. XHTML can't allow open tag,all tags must be closed.
All content can be included in the body element. All contents can be puts in blocks.
Attribute value are not significant in HTML. Attribute value are important in XHTML.
There is no hard rule for on the structure of the element. The structure of the element should be followed.
HTML was designed to display data and focus on how data looks. XHTML was designed to discribe data and focus on what the data is.
