Qu:-8 What are the various formatting tags in HTML?

Answer:
↣ There are various formatting tags in HTMl.
↣ Some formatting tags listed below.
  1. <b> ↣ To make text bold.
  2. <i> ↣ To make text ittalic.
  3. <u> ↣ To make text underlined.
  4. <mark> ↣ To make text marked.
  5. <sub> ↣ To make text subscript.
  6. <sup> ↣ To make text superscript.
  7. <del> ↣ To make text deleted.
  8. <big> ↣ To make text biger.
    ↣ This tag depricated from HTML-5.
  9. <small> ↣ To make text smaller.
  10. <strong> ↣ To make text bold.
    ↣ This tag depricated from HTML-5.
  11. <strike> ↣ To make text deleted.
    ↣ This tag depricated from HTML-5.
  12. <cite> ↣ To make text ittalic.
    ↣ This tag depricated from HTML-5.
  13. <emp> ↣ To make text ittalic.
  14. <ins> ↣ To make text underlined.
    ↣ This tag depricated from HTML-5.
  15. <br> ↣ To make line break.
  16. <hr> ↣ To create horizontal line in the page.
  17. <q> ↣ To write text in inverted comma("").
  18. <p> ↣ To create paregraph in the page.
  19. <code> ↣ To make text like code text.
  20. <pre> ↣ To create output like body written content.
  21. <center > ↣ To write text in center
    ↣ This tag depricated from HTML-5.
  22. <h1> to <h6> ↣ To create heading text in the page.
