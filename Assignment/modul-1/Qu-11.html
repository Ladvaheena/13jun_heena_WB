Qu:-10 What is the difference between a block-level element and an inline element?

Answer:
 
Difference b/w Block level element and Inline element
No Block level Element Inline Element
1 Block level element start with a newline. Inline/text level element do not start with a newline.
2 This element take up the full width avaiable. This element take up width as per the content.
3 This elememt accepts height and margins on all four sides. This element does not accepts height and top & bottom margin.
4 Block element like div tags are extensively used for defining layouts of the webpages. Inline element are used to give inline styles and effects.
5 Block element force a new line before and after they are used. Inline element do not force any new line.
6 This element contain text,data,inline element or other block level element. This element contain text,data or other inline element.
