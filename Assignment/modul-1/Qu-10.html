Qu:-10 How can we club two or more rows or columns into a single row or column in an HTML table? With Example.

Answer:
↣ There are two attribute used to club two or more row or columns into a single row or column in HTML table.
↣ Rowspan attribute used to club two or more row into a single row.
↣ Colspan attribute used to club two or more column into a single column.
