variable

-> variable are one type of container,in which we can store value.
-> In three ways,we can define variables
  1. Var
  2. Let
  3. Const

1. Var

-> Can Reinitialize & redefine also. means,we can change the value & redefine the variable.
-> For ex: 
    var a = 2 
        a = 5 --------- Reinitialize
    var a = 7 --------- Redefine

2.let

-> Can Reinitialize but can't Redefine again.
-> For ex: 
    let a = 2
        a = 6 -----------Reinitialize

3.Const

-> Can not Reinitilize & Redefine.
-> It is a constant Value. 
    Const a = 2
