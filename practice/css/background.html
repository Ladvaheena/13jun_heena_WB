Background image

↦ Background image apply in background property to pass image path in (url) in stytesheet.
↦ There are various property to make background more effective.
↦ here, we apply one background image in div tag,
↦ Now,we can set the size of image by using background-size perporty.
↦ if we do not want repeatation of image, we can set it by using background-repeat property& give value no-repeat.
↦ We can set the image position by using background-postion perporty.
↦ We can fix the position of the image by using background-attachment property & give value fixed.
↦ We can also apply simple background color by using backgroung-color property.
