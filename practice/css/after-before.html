After - Before Pseudoelement Selector

After & before Selector used to apply css after& before of the content.
In both selector, Content Property must be appiled.Without content property it's doesn't work.
In content Property,value must be apply in single coma ('').
After selecter apply by this way:(element::after{content:'value'}) in style sheet.
Before selector apply by this way:(element::before{content:'value'})in style sheet.
Advantages of this selector is that,we can apply css to all document remains in one element with one time applying css.
