Margin property

↦ In CSS, Margin property used to give space out side of the element content.
↦ In this property,we can use Margin top,bottom,left& right property.
↦ If we give only one value,it takes same space from all four sides.

↦ If We give two value,it takes top-bottom same space & left-right same space.
↦ If we give four value, it takes clock wise means (top,right,bottom,left)space.
↦ If we apply margin auto, it takes same space from right & left side.
