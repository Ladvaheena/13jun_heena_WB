☆ General selector

General selector used to select the group of element that share the common parent element.

This selector apply by element-1 tlide (~) symbol & element-2

☆ Adjacent Selector

The adjacent selector is used to select an element that is directly after another specific elements. Adjacent meaning Immediately following.
Sibling elements must have the same parent element,here parent element is body.
This selector apply by element-1 plus(+)symbol & element-2.
