Color Radient Property

↦ Color-radient CSS property is used create various color radient in web pages.
↦ This property only supported in background and background image property; its not supported in background-color property.
↦ We apply it by two ways,
  1. Linear Gradient
  2. Radial Gradient

✨ Linear Gradient

↦ It is used to apply(create) gradient in linear manner.
↦ We can also set the degree of the gradient by appling degree value in the property.

✨ Radial Gradient

↦ It is used to apply(create) gradient in radial(circular) manner.
↦ In this type,we can't apply the degree value.
