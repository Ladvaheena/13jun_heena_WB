Filter Property

↦ Filter property used to apply various show effect to an element like text,image or color .
↦ Syntax:img{fliter:fliter value(value in px or %)}
↦ This property has various value ,which gives various effect on image.which is listed below..
  1. Fliter:none---- do not show any effect; by default set value.
  2. Fliter:Blur----to show blurness in image; high value shows high blurness.
  3. Fliter:brightness---- to show brightness in image; 0% shows black,100% shows original image,>100% shows more brightness.
  4. Fliter:grayscale----to show gray scale color image; 0% shows original image,100% shows full Black & white image.
  5. Fliter:drop shadow----- to apply shadow to the image; 1st value for X-axis shadow,2st value for Y-axis shadow,3rd value gives spreadness 4th value for color.
  6. Fliter:hue-rotate----to apply according to hue color circle;value apply in deg.
  7. Fliter:opacity---- to apply transparency to the image; 0% shows full transparency means disappear the image in white background,100% shows original image.
  8. Fliter:sepia---- to apply yellowish color effect to the image;0% shows original image;100% is high value.
  9. Fliter:saturate---- to set the saturation of an image.100% shows original image;0%
  10. Fliter:invert---- invert the shade of color of an image.
  11. Fliter:contrast---- to apply contrast in color of an element.
