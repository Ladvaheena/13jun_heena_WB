background-origin property

↦ Background-origin property used to define the area, which is covered by background property.
↦ Background-origin property apply only on background-image poperty.
↦ This property has Three different value.
  1. Border box :- background image covers the contents & border area of the element..
  2. Padding box :- background image covers padding & content area of the element.it's by default value
  3. content-box :- background image covers only content area of the element.
