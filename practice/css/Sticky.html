Sticky Position property

↦ Position Property used to define the position of the element on webpages & website.
↦ Sticky-position property apply to the element,those element stick at the top part of the webpages,and other content scroll behind the stick element.
↦ In this type of position,also give the position,where we want to element to be stick,which means give value from top,right,left or bottom.
