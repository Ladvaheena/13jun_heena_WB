Background-clip property

↦ Background-clip property used to define area,which is covered by background.
↦ This property only apply on background-color property.
↦ This property has Three different value.
  1. Border box :- background color covers the contents & border area of the element.it's by default value.
  2. Padding box :- background color covers padding & content area of the element.
  3. content-box :- background color covers only content area of the element.
