⛋ Element selector

Element selector used to apply style in particular element.

⛋ Class selector

Class selector used to apply style in particular class name. Class selector apply with fullstop mark(.) & then written class name.

⛋ Id selector

Id selector used to apply style in particular id name. Id name must be unique. Id selector apply with hash symbol(#)& then written id name.

⛋ Universal selector ↣ Universal selector used to apply css to all web page.
