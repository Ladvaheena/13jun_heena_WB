Text Style Property

There are various text style property avaiable.With help of it,we make our content part more stylish & effective.list of it given below..
↦ Use of text align property,text set in center,left,right& justify.
↦ Use of color property, we can apply various color to the text.
↦ Use of font style property,text set in various style like; bold,italic etc...
↦ Use of font size property,increase & decrease the size of font .
↦ Use of word spacing property,apply space b/w two word in the content.
↦ Use of letter spacing property,apply space b/w two letter.
↦ Use of line-height property,apply space b/w two line.
↦ Use of text shadow property, apply shadoe to the text.
↦ Use of text decoration property,apply underline or overline to the text .
↦ Use of text decoration style property,apply various style to the underline.
↦ Use of text decoration thickness property,set the thinkness of the underline .
↦ Use of font weight property,make font bold.
↦ Use of text transform property, make all text lowercase,uppercase or capitilize.
↦ Use of text underline offset property,make space b/w text and underline.
