External css

External css apply by href attribute in link tag in head tag. In href attribute, pass the path of css file. Benefits of this type,we can apply style to web page through one file.

Internal Css

Internal css apply by style tag in head tag. Generally, css style apply to particular tag in curved brakets.

Inline css

Inline css apply in particular tag in style attribute. It gives output only those tag in which inline css style applied.
