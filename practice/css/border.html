Border Property

↦ Border is property which is used to apply border to content of the element.
↦ We can set border width,color ,style , radious etc...
↦ Set width of border with use of border-width property.
↦ Set color of border with use of border-color perperty.
↦ We can set different style of border with use of border style property.list of it given below..
  1. border-style:solid
  2. border-style:ridge
  3. border-style:dashed
  4. border-style:dotted
  5. border-style:inset
  6. border-style:outset
  7. border-style:groove
  8. border-style:double
  9. border-style:hidden for hide the border
  10. border-style:none -its by default set
↦ we can set all side different types of style with use of border top style,border bottom style,border left style and border right style.
↦ We can curve the edges of the border with use of border radiousproperty.
↦ We cal also set all four diffent curved edges.
