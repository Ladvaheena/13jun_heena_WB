Overflow property

↦ Overflow property used to define how the content show on webpages,if content nnot fitted into particular size element.
↦ it has various value to show content in different way..
  1. Overflow-visible :-remaining content visible on the pages.By default set this value.
  2. Overflow-scroll:- scroll line added in the content,reamaining content showed by scrolling.in scroll; if we apply overflow-x : scroll scroll bar only add in x-axis & content can visible from left-right. and if we apply overflow-y:scroll scroll bar only add in y-axis & content can visible from top-bottom.
  3. Overflow-hidden:- remaining content not visible, only shows content which is in element.
